# 반칠순 생존기록소 디자인 기준

이 문서는 현재 승인된 디자인을 잠그기 위한 기준이다. 이후 수정은 기존 시각 언어를 유지하며, 사용자가 명시한 변경 범위만 반영한다.

## 핵심 콘셉트

- 동네 잔치 포스터, 예능 자막, 오래된 가족 앨범의 혼합
- 반듯한 SaaS UI보다 손으로 붙인 종이·스티커·도장 질감 우선
- 웃기지만 생일 주인공을 조롱하지 않는 따뜻한 개그 톤

## 고정 디자인 토큰

- 잉크: `#251b1d`
- 종이: `#f8efdf`
- 핑크: `#f34f7f`
- 옐로: `#ffd742`
- 블루: `#405de6`
- 민트: `#78d7a5`
- 테두리: 2–4px 검정 실선
- 그림자: 검정 또는 섹션 포인트 컬러의 단단한 오프셋 그림자
- 제목: `Black Han Sans`
- 본문: `Gowun Dodum`

색상, 폰트, 굵은 외곽선, 종이 노이즈, 회전된 카드 스타일은 별도 요청 없이 변경하지 않는다.

## 레이아웃 규칙

- 최상위 섹션은 `100svh` 단위로 구성한다.
- 문서 전체는 `scroll-snap-type: y mandatory`를 사용한다.
- 각 섹션은 `scroll-snap-align: start`로 화면 시작점에 맞춘다.
- 모바일에서도 핵심 내용이 한 화면에 들어오도록 글자와 간격만 축소한다.
- 모바일 커서 조명은 숨기고 데스크톱 정밀 포인터에서만 표시한다.

## 승인된 화면 순서

1. 메인 반칠순 타이틀
2. 1992 사진 아카이브
3. 사건 개요 — 현장 증거 포함 한 화면
4. 방명록 작성 — 한 화면
5. 증언 리스트 — 한 화면, 페이지당 5개
6. 35세 사용설명서
7. 마지막 축하 및 연락 버튼

## 컴포넌트 유지 규칙

- 상단 캡슐 내비게이션 형태를 유지한다.
- 버튼은 굵은 테두리와 오프셋 그림자를 유지한다.
- 사진은 폴라로이드 카드, 테이프, 원형/사각 스티커 구성을 유지한다.
- 통계 카드는 핑크·옐로·블루·민트 순서를 유지한다.
- 방명록 카드의 회전, 컬러 그림자, 관계 라벨과 공감 버튼을 유지한다.
- 새 UI가 필요하면 기존 카드·버튼·라벨 스타일을 재사용한다.
